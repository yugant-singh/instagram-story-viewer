# Instagram Story UI Clone
# Live-Demo - https://instagram-story-viewer-three.vercel.app/

This project is a **mobile-style Instagram Story Viewer Clone** built
using **HTML, CSS and JavaScript**. It includes key Instagram-like
interactions such as:

-   Random front-view image on every refresh
-   Clickable story previews
-   Story viewer with auto-progress
-   Like/Unlike toggle (outline â†” filled)
-   Double-tap to like animation
-   Random comment count
-   Save/Unsave bookmark toggle
-   Center heart animation on like

### 1. Random Front View Image

Every time the page reloads, the main front image is randomly selected
from a list of image URLs.

### 2. Story Section (Horizontal Scroll)

-   Multiple story items displayed just like Instagram
-   Clicking any story opens it in a pop-up viewer
-   Progress bar runs automatically
-   Auto-close after 4 seconds

### 3. Story Viewer (Full View)

-   Shows one image at a time
-   Clicking advances to the next story
-   Automatically closes after progress completes

### 4. Like System

-   Click to toggle between:
    -   ri-heart-line (outline)
    -   ri-heart-fill (filled red)
-   Like count increases/decreases
-   Double-tap on image also likes
-   Beautiful center-heart animation

### 5. Save / Bookmark System

-   Click to toggle:
    -   ri-bookmark-line (outline)
    -   ri-bookmark-fill (filled)

### 6. Random Comment Count

Shows a random number (0--100) as comments count on page load.
